# E-Commerce REST API

RESTful API for an e-commerce platform using TypeScript, Express, and MongoDB.

Includes authentication, product and order management, and image uploads via @vercel/blob.

## Features

- JWT authentication
- Product & category CRUD
- Order processing
- File uploads

## Stack

- TypeScript
- Express.js
- MongoDB (Mongoose)
- Vercel Functions
